# Car-Price-Prediction Car Price Prediction using Machine Learning

## 📌 Project Overview

This project uses **Machine Learning** to predict the **current market price of a car**.
It helps both **buyers and sellers** make better and more efficient decisions.

---

## ⚙️ Libraries Used

* Pandas
* NumPy
* Matplotlib
* Seaborn
* Warnings

---

## 📂 Dataset

The dataset is collected from a CSV file and includes the following features:

* Brand
* Vehicle Age
* Kilometers Driven
* Mileage
* Engine
* Max Power
* Fuel Type
* Seller Type
* Seats
* Selling Price

Missing values are handled properly to ensure better prediction accuracy.

---

## 🧮 Model Process

1. Data preprocessing and cleaning
2. Feature selection
3. Splitting the data using **train_test_split**
4. Model building and training

---

## 🤖 Algorithms Used

* Linear Regression
* Decision Tree Regression
* Support Vector Regression (SVR)
* Random Forest Regression

Hyperparameter tuning is done using **RandomizedSearchCV**.

---

## 📊 Evaluation Metrics

* Mean Squared Error (MSE)
* R² Score (R2 Square)

---

## 🏆 Result

The best-performing model is selected based on accuracy and performance.

---

## 💾 Model Saving

The final model is saved using a **pickle file**, which can be used later for prediction.
